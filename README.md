# 🧠 Memory Card Matching Game

A fun and interactive **Memory Card Matching Game** built using **HTML, CSS, and JavaScript**.  
This project helps users improve memory skills by matching pairs of cards.

## ✨ Features

* 🧠 Card matching memory game  
* 🔀 Random card shuffle every time  
* 🎯 Match pair logic  
* 🎉 Winning alert (Hurray! You Won!)  
* 🔁 Auto restart after game completion  
* 🌫️ Blurred background image UI  
* 📱 Fully responsive design  
* 🎨 Clean and modern UI
